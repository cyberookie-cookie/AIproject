Technique ID: T1110.002
Name: Password Cracking

Description:
Attacker obtains password hashes and attempts to crack them offline using tools.

How it works:
- Acquiring hashes from SAM, shadow file, LSASS, or DB.
- Using brute force or dictionary attack offline.

IOC:
- Access to credential stores.
- LSASS dumping events.
- Suspicious tools like hashcat, john.

Log Sources:
- Sysmon Process Create
- Windows Event ID 4656 (handle access)
- Linux sudo logs

Detection:
- Detect LSASS memory access.
- Alert on use of known cracking tools.

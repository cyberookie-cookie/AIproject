Technique ID: T1110.001
Name: Password Guessing

Description:
The attacker manually or automatically tries multiple password combinations for a specific user account.

How it works:
- Focuses on a single target account.
- Tries common passwords or dictionary words.
- Slow but targeted approach.

IOC:
- Multiple failed login attempts for one user.
- Authentication attempts at unusual hours.
- Repeated failures with slightly different passwords.

Log Sources:
- Windows Event ID 4625
- Linux auth.log
- SSH logs
- Web server login logs

Detection:
- Threshold alert for repeated failures on one username.
- Detect login attempts from unusual IP addresses.

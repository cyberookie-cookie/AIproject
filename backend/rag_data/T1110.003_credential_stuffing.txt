Technique ID: T1110.003
Name: Credential Stuffing

Description:
Attackers use leaked username/password combos from previous breaches to log in.

How it works:
- Large lists of known credentials.
- Automated login attempts on many accounts.
- Often uses rotating IPs.

IOC:
- Many accounts targeted from same IP.
- Logins using known leaked credentials.
- High authentication failure rate.

Log Sources:
- Web server logs (HTTP 401)
- Windows 4625
- Linux SSH logs

Detection:
- Correlate failures across many accounts.
- Detect logins from IPs known in breach databases.
